Grade Notifier Bot

// Automated grade monitoring · Real-time notifications

Python Selenium Tkinter SMTP Automation
// Description

Automated grade monitoring bot that periodically checks the BGU student portal, detects newly published exam results, downloads the exam files, and notifies the user in real time via both a desktop sound alert and an email notification.

Runs silently in the background using a lightweight Tkinter tray interface. Built with Selenium for portal authentication and DOM scraping, with SMTP integration for the email delivery pipeline.

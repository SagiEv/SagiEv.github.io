// Overview

Full-stack e-commerce platform built end-to-end — from a responsive React UI to scalable Spring Boot REST APIs backed by a relational PostgreSQL data model.

The project focused on modular architecture for long-term maintainability, clean separation of concerns across layers, and real-world team collaboration via structured GitHub workflows.

// Key Features
  • Product catalog with filters, search, and pagination
  • JWT-based authentication and role-based authorization
  • Order lifecycle management from cart to checkout
  • RESTful API with full CRUD operations on all entities
  • Responsive UI with component-based React architecture
// Challenges & Solutions
CHALLENGE
Keeping the frontend and backend in sync as the data model evolved during development
SOLUTION
Defined shared DTOs as the contract boundary early, versioned API changes via feature branches with PR reviews
CHALLENGE
Handling concurrent order submissions without leaving inventory in an inconsistent state
SOLUTION
Applied optimistic locking on inventory entities and wrapped order creation in a transactional service layer

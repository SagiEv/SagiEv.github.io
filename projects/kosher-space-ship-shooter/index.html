Kosher Space Shooter

// Sound Design & Game Audio

Reaper Processing Foley Recording Digital Synthesis
// Description

Designed and implemented the full sound design for a sci-fi shooter game — covering every audio layer: laser fire, explosions, ship engine hum, enemy death sounds, and UI feedback clicks.

Combined digital synthesis techniques in Reaper with real-world Foley recordings (metal impacts, plastic textures, organic scrapes) to create immersive, layered audio that felt both futuristic and grounded. The game audio was built to react dynamically to gameplay events, reinforcing player feedback through sound timing and intensity.

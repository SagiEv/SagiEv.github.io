// Noa Kirel Filter - Detailed Creation Process

I created this Instagram filter inspired by Noa Kirel by combining 3D modeling, animation, and AR tools into one workflow.

First, I gathered reference images of Noa to capture her facial features accurately. Using Blender together with the FaceBuilder plugin, I built a realistic 3D model of her head.

For the body, I started with a free-to-use 3D character model, then modified it to match the proportions and style I wanted. I carefully attached the custom head to the body and refined everything so it looked seamless.

I also modeled the lollipop prop from scratch in Blender, and created all the textures myself using Adobe Photoshop to give it a polished, stylized look.

To bring the character to life, I used Mixamo — an online tool by Adobe that automatically rigs 3D models (adds a skeleton) and lets you apply ready-made animations without needing to animate manually. This made it easy to create smooth, dynamic movement for the character.

Finally, I imported everything into Spark AR Studio, where I built the AR environment, added music, and set up the filter so it could be placed and experienced directly in Instagram.

The filter ended up going viral, and was even used by Ran Danker and Shachar Tavoch — and Noa Kirel herself shared it, which was an amazing moment.

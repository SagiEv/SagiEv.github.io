Instagram AR Filters

// Real-time Face Tracking · 3D Modeling · Spark AR

[ SCREENSHOT / DEMO VIDEO ]
Spark AR Blender FaceBuilder Photoshop Mixamo
// Description

Developed a set of interactive Instagram AR filters using real-time face tracking, 3D modeling, and skeletal animation. Filters included celebrity-inspired transformations and a fully personalized digital avatar built from scratch.

The pipeline covered the full production cycle: 3D face mesh creation in FaceBuilder, model rigging and animation in Blender and Mixamo, texture painting in Photoshop, and final integration and publishing in Spark AR — unifying rendering, real-time interaction, and user experience design into a single deployable filter.

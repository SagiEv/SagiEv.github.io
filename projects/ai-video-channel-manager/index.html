// Description

A centralized dashboard for creating and managing multiple AI-generated video channels. Users can spin up new channels or trigger video production for existing ones — all from a single React interface.

Each action fires an n8n workflow via API that handles the full production pipeline: prompt generation with Gemini, video generation via Sora, and a Telegram-based human approval step before the video is published to the target channel.

Built with Claude Code as an AI-assisted development tool, showcasing a modern agentic development workflow alongside traditional full-stack engineering.

// Approval Flow Demo

1. The Trigger

The production begins in the TubeFlow dashboard. Clicking Generate Video fires a webhook to n8n, initiating the multi-agent creative process.

2. Human-in-the-Loop

To ensure quality, the final prompt and video preview are sent to Telegram. I can approve or decline the post with a single tap, keeping the flow automated but supervised.
